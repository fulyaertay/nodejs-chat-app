## ðŸ”‹ Installation

**1. Clone the Repository**

```
  git clone https://github.com/fulyaertay/nodejs-chat-app
```

**2. Move into the working directory**

```
  cd nodejs-chat-app
```

**3. Install Dependencies**

```
  npm init -y
  npm i express socket.io
```


**4. Run the App**

```
   nodemon server.js
```
